# 📁 Panduan Memindahkan File ke htdocs

Berikut adalah langkah-langkah memindahkan file ke folder `htdocs` agar dapat diakses melalui localhost:

## 🚀 Langkah-langkah

1. **Buka XAMPP**
   - Jalankan aplikasi XAMPP.
   - Pastikan Apache dan MySQL dalam keadaan **Start** (ditandai dengan warna hijau).

2. **Akses Folder htdocs**
   - Buka File Explorer.
   - Navigasikan ke direktori tempat XAMPP terinstal, biasanya:
     ```
     C:\xampp\htdocs
     ```

3. **Pindahkan File atau Folder Proyek**
   - Salin file atau folder proyek Anda.
   - Tempel ke dalam folder `htdocs`.

4. **Cek di Browser**
   - Buka browser favorit Anda.
   - Akses proyek Anda melalui URL berikut:
     ```
     http://localhost/namafolderproyek
     ```

## 🧪 Contoh
Misalkan Anda memiliki folder proyek bernama `websiteku`, Anda bisa mengaksesnya melalui:
```
http://localhost/websiteku
```

## ❗ Catatan Penting
- Pastikan file utama (biasanya `index.php` atau `index.html`) ada di dalam folder proyek.
- Jika ada error, cek kembali apakah Apache dan MySQL sudah berjalan di XAMPP.

## ✅ Selesai!
Sekarang file Anda sudah bisa diakses melalui localhost. 🎉
